> High-performance TradingView webhook signal router for multi-broker order execution. Supports paper trading, risk management, and real-time order lifecycle tracking.

## ✨ Features

- ✅ High-performance TradingView webhook signal router for multi-broker order execution. Supports paper trading, risk management, and real-time order lifecycle tracking.

## 📦 Installation

```bash
git clone https://github.com/DinhLucent/tradingview-signal-router.git
cd tradingview-signal-router
pip install -r requirements.txt
```

## 🚀 Quick Start

```bash
python src/main.py
```
